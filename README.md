## Commands

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

Run Prettier:

```bash
npm run prettier
```

Run ESLint:

```bash
npm run lint
```
